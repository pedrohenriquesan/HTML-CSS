Faça uma escolha

A ideia deste tópico é, como navegar por diferentes páginas em um mesmo iframe declarado, antes de explicarmos o modo de fazer, é importante dizer que é interessante separar as páginas do iframe uma outra pasta no seu projeto..

  • Primeira Página
  • Segunda Página
  • Terceira Página

Para fazermos essa ação possível utilizamos os links, no iframe é obrigatório a declaração de um “name” para o iframe, somente com ele funcionará, enquanto isso, no atributos target das nossas âncoras colocamos como valor o “name” dado ao iframe, com isso, todas as páginas irão abrir em um mesmo iframe sem sair da página principal.
