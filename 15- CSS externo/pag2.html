Por conseguinte, a tag link é criada com dois parâmetros, rel com valor padrão stylesheet e o href que recebe o nome do documento .css, com isso, basta criar o arquivo css e linkar o mesmo as páginas html desejadas pelo parâmetro href. Com isso, quando for necessária uma modificação basta modificar o arquivo css e todas as páginas linkadas a ele vão se personalizar com a mudança.
