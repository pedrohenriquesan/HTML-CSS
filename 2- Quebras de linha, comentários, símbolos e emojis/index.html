Quebras de linha, comentários, símbolos e emojis

Para fazer quebras de linha não basta apenas apertar enter como fazemos no dia a dia, para isso existe a tag <br> que faz a função comum do enter
a tag br quebra a linha.

Comentários em HTML são feitos do seguinte modo: <!--COMENTÁRIO-->

Veja alguns símbolos (os valores entre parenteses indicam o nome de cada símbolo, todos após o prefixo (&) e antes do sufixo (;)):
<: sinal de menor (lt)
>: sinal de maior (gt)
 : espaço em branco (nbsp)
®: marca registrada (reg)
©: copyright (copy)
€: euro (euro)
£: libra(pound)
¢: cent (cent)
∅: vazio (empty)
∑: soma (sum)
Δ: delta maiúsculo (Delta)
δ: delta minúsculo (delta)
←: seta para esquerda (larr)
↑: seta para cima (uarr)
→: seta para direita (rarr)
↓: seta para baixo (darr)

Veja alguns emojis (cada emoji possui um código e, esse código vem depois do prefixo &#x e antes do sufixo (;)):
😂: código (1F602)
🥰: código (1F970)
🤑: código (1F911)
