CSS

CSS interno

Criamos a tag style na área head do documento HTML, com isso, tudo que for escrito nessa área será considerado estilo. Esses estilos são escritos por meio de declarações (conjunto de característica e valor terminada por ponto e vírgula), as quais são acomodadas por seletores (chaves), ou seja, enquanto no HTML usamos os colchetes angulares, aqui usamos os seletores. A grande vantagem desse método é que a estilização não é feita somente em uma área pontual, por exemplo, no CSS inline editamos um h1 em específico (área pontual), no CSS interno editamos todos os h1 de forma simultânea (área não pontual), em resumo, criamos um seletor para uma tag (h1) ou área (body) e todas as tags ou áreas daquele tipo vão adquirir as características dessas declarações.
