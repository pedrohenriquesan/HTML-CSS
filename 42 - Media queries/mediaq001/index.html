O que são media queries

Atualmente, existem muitos dispositivos de tamanhos diferentes de tela, e é para isso que as media queries servem, cada media query se refere a um formato de site para cada tipo de dispositivo do usuário, ou seja, o site deve ter diferentes versões de estilo para que o usuário consuma aquela conteúdo sem nenhum problema, geralmente um site completo tem várias media queries, uma para cada dispositivo, como televisões robustas, computadores, tablets, celulares e até impressoras.

Media types

O que vamos ver a seguir são chamadas de media types, ou seja, diferentes formas de portar o seu site para diferentes formatos de mídia, no exemplo dado abaixo, temos um site simples para um computador comum e para impressora (como um site de notícias), para isso, usamos dois medi types, screen para telas e print para impressão. Existem outros tipos de media types, inclusive para impressão em bryle, mas vamos usar somente screen, print e all.
