Tabelas maiores

 Quando o assunto são tabelas com muitos dados, o próprio HTML indica o uso de tabelas com tags semânticas, análogas a que já conhecemos para o conteúdo, como as tags main, section, article e footer. As tags semânticas que vamos apresentar para as tabelas são:
⇒thead, área que se encontra o cabeçalho de uma tabela.
⇒tbody, área que se encontra todos os dados de uma tabela.
⇒tfoot, área que se encontra o rodapé de uma tabela.

 Quando temos tabelas muito longas em dados e colunas fica difícil a visualização entendimento dos dados sem saber o seu tipo presente no cabeçalho, por isso é importante fixarmos o cabeçalho de uma tabela, mas, é importante ressaltar que o modo exemplificado abaixo não funciona em todos os navegadores, use com cuidado.

É necessário as seguintes declarações para fixar o seu cabeçalho:
⇒a sua tabela deve assumir uma posição relativa, position = relative;
⇒os títulos presentes no seu thead deve assumir posição “grudada” na posição 0 acima, position = sticky; top 0;
⇒as configurações de estilo acompanham as declarações acima para os títulos.
