Tabelas

É que é EXTRAMAMENTE contraindicado a utilização de tabelas para criar a estrutura do seu site, antigamente, no HTML4, eram usadas tabelas para construir o layout dos sites, mas isso não pode ser usado mais, é ruim em vários sentidos, as tabelas são muito rígidas e levam uma hierarquia de organização muito única e, como já sabemos, nossa estrutura deve ser maleável para atender todos os tamanhos de tela, se os detalhes de uma tabelas forem violados seu site certamente irá cair. Resumidamente, não use, em nenhuma hipótese para a estrutura do seu site, mas para outras ocasiões, as tabelas podem ser a opção perfeita.

As tabelas são estruturas que possuem linhas e colunas juntamente com seus dados que são identificados pelas linhas e colunas nas quais se encontram. Vamos ao HTML.
