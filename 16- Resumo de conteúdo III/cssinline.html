CSS inline

O CSS inline é o primeiro modo de se criar folhas de estilo, é criado no próprio documento HTML, todas as declarações de estilo são colocadas dentro do parâmetro style no conteúdo correspondente, além disso, todas essas declarações são feitas em uma mesma linha (inline), tal método não é indicado normalmente, uma vez que, causa muita poluição visual no código HTML, ele deve ser usado como última opção em modificações repentinas e básicas.
