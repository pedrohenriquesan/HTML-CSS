Modelo de caixas

Um dos assuntos mais importantes para o desenvolvimento web é o modelo de caixas, tal teoria é aplicada no HTML e no CSS e é responsável por auxiliar na organização do design do site. Como o nome já indica, o modelo de caixas está ligado literalmente a caixas, onde todo e qualquer conteúdo de um site se encontra dentro de uma caixa, inclusive quase todas essas tags que viemos utilizando, além disso, caixas podem ser inseridas dentro de caixas criando uma hierarquia, tal hierarquia recebe o nome de aninhamento.

Só para exemplificar, os links são caixas do tipo inline level Acesse meu Gituhub. Enquanto os títulos de nível 1 (h1) e os parágrafos são caixas do tipo box level. Veja como eles se comportam na viwport na próxima página.

Box level

As caixas box-level, são aquelas que sempre se iniciam em uma nova linha e ocupam a largura completa da viewport (tela). O exemplo mais famoso é a div.

Inline level

As caixas inline-level, são aquelas que não iniciam em uma nova linha quando há espaço e ocupam apenas o espaço necessário para o conteúdo da caixa. O exemplo mais famoso é a span.

Propriedades Display

É importante dizer que é possível alterar o formato de uma caixa, ou seja, é possível tornar uma caixa box level em inline level e vice versa, tal ação é muito poderosa para os desenvolvedores e é feita pela propriedade display, onde os valores block, inline e inline block significam respectivamente: box level, inline level e inline level com algumas propriedades box level como as declarações de altura e largura.

Anatomia das caixas

As partes de uma caixa são:

  • conteúdo → conteúdo em si.
  • width → largura da caixa.
  • height → altura da caixa.
  • padding → espaço interno entre o conteúdo e a borda da caixa, ou seja, por padrão, a borda (border) fica grudada com o limite do conteúdo da caixa e o padding é responsável por aumentar essa distância conteúdo-borda, tal distância pode ser chamada de acolchoamento ou preenchimento.
  • border → borda da caixa.
  • outline → tracejado localizado do lado de fora da borda na região do margin, ou seja, também chamado de contorno ou traçado, o contorno é fora do elemento enquanto a borda é dentro do elemento, mas é pouco utilizado.
  • margin → espaço externo entre a caixa e outros elementos fora da caixa, ou seja, além do espaço interno padding, também é possível alterar o espaço externo da caixa, uma vez que, o padrão faz com que os elementos de fora da caixa ficam “grudados” na caixa, com isso, para que a borda da caixa não cole nos conteúdos a sua volta use a propriedade margin.
