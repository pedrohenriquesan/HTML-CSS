Links - primeira página

Links externos

Links externos são aqueles que direcionam a navegação para páginas externas ao próprio site, ou seja, outros sites.
Exemplos:

Acesse meu perfil no github clicando aqui.

Acesse o site de referência da linguagem HTML clicando aqui.

Links internos

Links internos são aqueles que direcionam a navegação e interação dentro do próprio site, ou seja, eles são responsáveis pelas transições de telas(páginas) de um mesmo site. Exemplo:

Clique aqui para ser direcionado para a nossa segunda página!

Links internos em subpastas

A única diferença presente aqui é o local do arquivo html, estando em uma subpasta junto ao index, onde a referência hipertexto é feita do mesmo modo que as imagens, veja:

Clique aqui para acessar nossa terceira página!
