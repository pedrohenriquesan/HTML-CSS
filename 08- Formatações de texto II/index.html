Código fonte

Tag code

A tag code é responsável por tornar a fonte do texto monoespaçada, além de indicar que aquele trecho se trata de um código fonte.

→Exemplo de um código na linguagem C sem a tag code (declaração e inicialização de vetor): int vetor[3] = {0, 1, 2}

→Exemplo de um código na linguagem C com a tag code (declaração e inicialização de vetor): int vetor[3] = {0, 1, 2}

Tag pre

A tag pre basicamente mostra o texto do mesmo modo que ele foi digitado no seu editor de código.

→Exemplo de um código da linguagem C sem a tag pre (função principal da linguagem):

int main () { printf ("Função principal da linguagem C!"); return 0; }

→Exemplo de um código da linguagem C com a tag pre (função principal da linguagem): 

                                                                                                   
            
    int main ()
    {
            printf ("Função principal da linguagem C!");

            return 0;
    }

Citações

Tag q

A tag q é usada para citações simples que aparecem no meio de parágragos, seu resultado visual é basicamente as aspas.
Exemplo: Como diria o tio ben: com grandes poderes vem grandes responsabilidades.

Tag blockquote

A tag blockquote é usada para citações mais completas, seu resultado visual é um espaçamento da citação a direita em relação a margem do site. Veja:
Segundo o autor X, no livro Y:

Existem tags em HTML que são semânticas e não semânticos, ou seja, possuem e não possuem significado respectivamente.

Abrevições

Tag abbr

A tag abbr é responsável por indicar através do parâmetro title o que significa determinada sigla, o resultado visual aparece ao correr o cursor sobre a sigla.
