Flexbox

Conceitos básicos

Flexible Box Module ou Flexbox é um modelo de layout unidimensional capaz de organizar elementos em uma interface com alta capacidade de alinhamento, todas as operações com flexbox relacionam-se com dois eixos, o principal e o transversal, o principal é definido pela propriedade flex-direction (mais informações na seção sobre as propriedades), enquanto o transversal se encontra perpendicular a ele, compreender a diferença entre os eixos é o que importa quando começamos a observar o alinhamento ou justificação dos itens flexíveis (flex-items), o flexbox possui propriedades que alinham e justificam o conteúdo ao longo de um eixo ou de outro.

Contêiner flex

A área de um documento que faz uso do flexbox é denominado contêiner flex, para criar essa estrutura definimos a propriedade display assumindo como valor flex ou inline-flex, desse modo, todos os elementos filhos vão assumir comportamento flex, com isso, algumas características serão aparentes, além das seguintes propriedades assumindo seus valores padrões:

  • flex-direction assume row;
  • Alinhamento na borda do eixo principal;
  • Sem expansão do eixo principal, mas talvez contração;
  • Expansão vertical para preencher a altura do eixo transversal;
  • flex-basis assume auto;
  • flex-wrap assume nowrap;

Por fim, o resultado final dessa configuração padrão, é que todos os elementos vão ser alinhados em uma única linha, usando o tamanho do conteúdo como tamanho do eixo principal, se houver mais itens do que é possível não há quebra de linha e eles irão ultrapassar a tela, todos os itens se estendem pelo eixo transversal até a altura do maior item.

Propriedades

display

flex-direction

Indica a direção na qual os elementos serão exibidos e pode receber os seguintes valores: row (elementos dispostos em ordem na linha da esquerda para direita, é o padrão), row-reverse (elementos dispostos em ordem na linha da direita para esquerda), column (elementos dispostos em ordem na coluna de cima para baixo) e column-reverse (elementos dispostos em ordem na coluna de baixo para cima). TESTAR

flex-wrap

Apesar de ser um modelo unidimensional o flexbox permite que elementos sejam agrupados em múltiplas linhas, com isso, cada linha é considerada um novo contêiner flex, para isso, usamos a propriedade flex-wrap par quebra automática de linha quando uma for ocupada totalmente, tal propriedade pode assumir dois valores: nowrap (não quebra a linha, é o padrão) e wrap (quebra a linha). TESTAR

flex-flow

É uma shorthand para as duas propriedades anteriores, a declaração flex-flow deve assumir o valor de flex-direction seguido do valor de flex-wrap.
