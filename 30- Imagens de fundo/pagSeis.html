Teste de imagens de fundo

Uma dica poderosa é, geralmente, muitos podem pensar que se o seu site tem uma imagem de fundo não é necessária uma cor de fundo, mas é sim, pois o conteúdo é carregado primeiro que a imagem, por isso, é ideal cor de fundo inicialmente. Para a propriedade background-size é muito bom usar o valor de 100%. Mas, em na grande maioria das ves sobra-se uma parte em preto embaixo, é possível concertar configurando o height do body para 100vh, e 100% de largura e altura no seu background-positon, mas isso irá distorcer as imagens.

Para resolver essa solução ruim existe os valores "escritos” para o background-size:
- Valores numéricos, porcentagens são mais indicadas.
- Auto, opção automática, aplica a imagem o tamanho original.
- Contain, dá o comando que a imagem seja 100% exibida, sem cortes, essa não resolve nosso problema.
- Cover, cobre toda a tela sempre, mas corta as imagens.
