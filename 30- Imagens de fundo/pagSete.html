Sempre que for centralizar verticalmente, é necessário a presença de um contêiner e um conteúdo. Como vimos antes, é relativamente simples centralizar o conteúdo de um contêiner horizontalmente com o comando margin: auto, mas quando falamos em alinhamento vertical isso não é possível. Para isso vamos usar os posicionamentos, por padrão, é necessário que o contêiner possui posicionamento relativo (padrão), enquanto o conteúdo deve ter posicionamento absoluto, quando essas propriedades são definidas, duas propriedades podem ser colocadas para o conteúdo absoluto:
→left
→top
Assumindo o valor de 50%, dá a impressão que o problema seria solucionado, mas o ponto de alinhamento se dá no ponto superior esquerdo, não ficando no centro.

Para isso vamos usar uma nova propriedade, a propriedade transform e seu valor translate, essa propriedade é mexer em qualquer caixa, em vários meios disponível, a operação translate mexe no posicionamento valores positivos vão para direita e para baixo, negativos vão para esquerda e para cima. Com isso, basta usarmos essa propriedade com valores negativos de 50% negativos.
