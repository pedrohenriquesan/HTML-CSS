Formatações de texto

Veja neste parágrafo algumas das principais formatações de texto vistas, lembrando que essas formatações alteram a forma (estilo), mas também possui um significado semântico, por isso, é importante sempre se atentar as tags que possuem e não possuem significado (semânticas e não semânticas). Por fim, lembre-se que o HTML se trata do conteúdo, enquanto o CSS trata do estilo. Foram usadas as tags strong para negrito, em para itálico, ins para texto sublinhado, del para texto excluído e abbr para as abreviações.

Links

Os links podem ser externos ou internos, os externos ligam seu site a um outro, enquanto os internos ligam uma página do seu site a outra do mesmo.
Exemplo externo: Clique aqui para acessar meu perfil no Github!
Exemplo externo: Clique aqui para acessar o Youtube!
É importante dizer que para a abertura ocorrer em uma nova aba o parâmetro target é necessário com valor _blank e rel com valor external indicando para o navegador que se trata de um site externo ao seu.
