CSS inline

As CSS inline são feitas em áreas pontuais no próprio documento HTML, ou seja, os estilos são adicionados área por área dentro do parâmetro style do HTML, ademais, um conteúdo recebe todas as suas configurações de estilo em uma mesma linha, além disso, se você possui dois conteúdos h1 por exemplo, a mesma estilização deve ser inserida em ambos, uma vez que, as estilizações são pontuais e não levam em consideração todos os h1. Nessa página, por exemplo, foram alteradas as seguintes características:

  • background-color altera a cor de fundo da página.
  • color altera as cores do texto.
  • font-family altera a fonte.
  • font-size altera o tamanho da fonte.
  • text-align indica o modo de organização do parágrafo.

Observação: essas "características" são inseridas dentro do parâmetro style nas áreas pontuais, desse modo:
style="background-color: cor; color: cor;..."
